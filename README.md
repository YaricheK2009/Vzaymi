# Взаймы

Простой сайт объявлений про аренду на Python. По смыслу это как доска объявлений, но вместо продажи здесь аренда.

## Что умеет

- показывает список объявлений;
- фильтрует по запросу, категории и городу;
- открывает отдельную страницу объявления;
- добавляет новое объявление через форму;
- хранит данные в SQLite;
- регистрирует пользователей по почте и паролю без одноразовых кодов.

## Как запустить

```bash
python app.py
```

После запуска открой в браузере:

```text
http://127.0.0.1:8000
```

Для демо-пользователей из сидов пароль один и тот же:

```text
demo12345
```

## Файлы

- `app.py` — сервер, маршруты и работа с базой;
- `static/styles.css` — оформление;
- `rentify.db` — база данных, создается автоматически при первом запуске.
